# Tone Language

Use these as conservative directional heuristics, not fixed recipes.

| User language | First adjustments |
| --- | --- |
| Too bright / harsh / fizzy | Reduce presence or treble slightly; reduce high-cut frequency only after checking cabinet choice; reduce drive gain before heavy EQ. |
| Too dark / muffled | Reduce excessive bass/low mids; raise presence or treble slightly; shorten dark delay/reverb tails. |
| Muddy / boomy | Reduce amp bass, drive low end, or reverb low-frequency content; avoid increasing treble as the only fix. |
| Thin | Add a small amount of low-mid body, reduce excessive high cut, or choose a fuller cabinet; do not solve with output volume. |
| More attack / clarity | Reduce compression and gain, shorten ambience, and preserve upper mids. |
| Smoother lead | Add moderate sustain, reduce harsh presence, use controlled delay and reverb, and keep output changes conservative. |
| Wider / ambient | Prefer stereo modulation/delay/reverb when available; increase mix in small steps and protect dry-note definition. |
| Tighter metal rhythm | Reduce preamp low end, use a low-gain overdrive boost, moderate amp gain, controlled gate, and short ambience. |
| Clean Japanese pop rhythm | Low-to-moderate compression, articulate clean amp, restrained chorus, short delay, and low reverb mix. |

Ask about guitar/pickups, monitoring path, tuning, and musical role only when those facts materially change the plan. Prefer one concise question at a time.
